# bp-document-status

Статус операционных документов для **1С:Бухгалтерия** (расширение): перечисление статусов, роль, блокировка при «Выполнено», защита при обмене с КА.

## Документы

- `РеализацияТоваровУслуг`
- `ПоступлениеТоваровУслуг`
