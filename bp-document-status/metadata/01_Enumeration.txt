Перечисление: СтатусыДокументовОпераций
=====================================

Конфигуратор → Перечисления → Добавить.

Имя (англ. идентификатор): СтатусыДокументовОпераций
Синоним: Статусы документов операций

Значения перечисления (порядок важен для отображения по желанию):
1. ВРаботе     — синоним: «В работе»
2. ВДоработке — синоним: «В доработке»
3. Выполнено  — синоним: «Выполнено»

Значение по умолчанию для новых документов в коде: Перечисления.СтатусыДокументовОпераций.ВРаботе
(см. модуль объекта — ОбработкаЗаполнения / вызов из общего модуля).

Если переименуете перечисление или значения — синхронно поправьте ссылки в
CommonModules/ДокументыСтатусыОпераций/Module.bsl (или переименуйте только
после замены во всех вставках).
